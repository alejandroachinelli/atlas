# Atlas — Backend Architecture Portfolio (.NET + Cloud + IA)

Atlas es un proyecto de portfolio diseñado para demostrar arquitectura moderna en .NET:

- Backend modular (monolito modular evolucionable a microservicios)
- Seguridad moderna (JWT clásico vs OIDC vs BFF)
- Observabilidad (logs, métricas, trazas, SLOs)
- Cloud (Azure-first, multi-cloud awareness)
- IA aplicada (RAG, agentes, vector DB)
